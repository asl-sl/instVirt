instVirt(Instrutor Virtual).

Este projeto tem a finalidade de auxiliar professores em sala de aula.

Trata-se de um pequeno aplicativo em python que permite a integração de ambiente GUI com linguagem python a textos pre estabelecidos e sua exibição.

Atualmente desenvolvido como um estudo da Academia do software livre, um projeto pessoal para divulgar o uso de software livre em nossa região, (sul de Minas Gerais).

Nesta versão inicial ele traz alguns textos de exemplo que já possuem um conteúdo didático aplicado a ciência da computação.
